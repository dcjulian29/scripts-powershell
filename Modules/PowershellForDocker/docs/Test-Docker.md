# Test-Docker

## SYNOPSIS

Test if the docker.exe is present.

## SYNTAX

```powershell
Test-Docker
```

## DESCRIPTION

The Test-Docker function test to see if docker.exe is present on the system.

## EXAMPLES

### Example 1

```powershell
PS C:\> Test-Docker

True
```

This example returns true because docker.exe is installed.

## PARAMETERS

### None
