# Update-Profile

## SYNOPSIS

Reload the profile script.

## SYNTAX

```powershell
Update-Profile [<CommonParameters>]
```

## DESCRIPTION

The Update-Profile cmdlet reloads the profile script into the current session.

## EXAMPLES

### Example 1

```powershell
PS C:\> Update-Profile
```

This example reloads the profile script in the current session.
